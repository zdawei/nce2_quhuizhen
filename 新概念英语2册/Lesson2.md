# 单词

## until 
1. prep. （介词）直到。。。 (介词后面跟名词代词等)
2. conj. （连词）直到。。。 (连词后面跟句子)

## not...until... 
到。。。时才/再。。。

## from...until
从。。。时候到。。。时候

## till
等同于 untill

## up to
直到。。
由于to是一个介词，所以up to是个介词短语，后面跟名词或者代词

## 到现在为止
so far = until/till now = up to now

## so far, so good
到现在为止，一切都好

# 课文

## it 
虚主语多指时间，天气，环境，距离。

## 时间频率副词 
never / rarely / seldom / sometimes / often / usually / always …副词在句中多位于实义动词（表示具体动作）之前, 助动词（do / be / have …）和情态动词（can / may / must / should / will…）之后。

I sometimes stay in bed until lunchtime.
You should never trust him.

在疑问句中多位于主语之后。
Do you always get up so late?

## sometimes

sometimes: 有时
some times： 几次
sometime: 某时
some time: 一段时间

## out of
out (of) the window 向 / 在窗外

out (of) the door 向 / 在门外

括号里的of 可以省略

## 感叹句

1. what + a / an (+形容词 ) + 可数名词单数 (+主+谓 ) !

    感叹句，括号里可省略

2. what +形容词+可数名词复数 / 不可数名词 +(主+谓 )!
    
    形容词，不可省略

3. how +形容词/副词+(主语+谓语)!
4. how +形容词+ a / an +可数名词单数 +(主+谓)！

## 现在进行时
### 语法
1. 肯定句: Subject+ be (am / is / are) + doing.
2. 否定句: S. + be + not + doing
3. 一般疑问句: be + S. + doing?
4. 特殊疑问句: 疑问词 + be + S. + doing?
### 意思
1. 表示正在进行的动作
    
    But I’m still having breakfast.

    What are you doing?

2. 表示即将发生的动作

    I’m going.

    He’s dying.

## 对没有看见的人应该用 it
Who is it?

It’s John / Mary.

## 名字
1. 名: first name / given name / Christian name
2. 姓: last name / family name / surname
3. 名在前姓在后 Lucy White
4. aunt Lucy

## just
1. just adv. 刚刚（句中位置同时间频率副词）
2. 英式用于现在完成时
3. 美式用于一般过去时。 I just arrived by train.

## 固定搭配

1. by train / bus / car / ship / boat / plane …
2. on foot
3. have breakfast / lunch / supper / dinner （只能使用have，实义动词）
4. have / eat + fish / chicken / beef / soup … 吃具体的东西，可以用eat或者have
5. take medicine / a pill 吃药用take
