# friendly 
friendly  adj. 友好的，亲切的

n. + ly 形容词 “像……一样的"

- motherly 母亲般的
- manly 有男子气概的，像男人的
- womanly 有女子气质的，像女人的
- miserly 吝啬的 (miser n. 守财奴)
- cowardly 胆小的 (coward n. 懦夫)

# whole
adj. 整个的，全部的

用法：之前一般需有冠词，之后常不跟复数名词。 
- e.g. a whole cake
- e.g. the whole day

同义词：
entire 意思用法与 whole 基本一样

e.g. the entire family

the entire human race

all 意思与 whole 一样, 用法略有不同

1. 可以不要冠词，若需要只用 the，置
于 all 之后。
2. 可修饰单数名词也可修饰复数名词。

e.g.
- all day
- all morning
- all night
- all the time 一直…
- all the class
- all the students

# 一般过去时态

使用场合
1. 表示过去某时发生的动作或存在的状态。 
2. 过去某段时间内的习惯，反复发生的动作

结构
1. 一般动词：
 - 肯定：S + 动词过去式
 - 否定：S + didn’t + 动词原形
 - 疑问：Did + S + 动词原形…?

2. be 动词
- 肯定：S+ was/were…
- 否定：S + wasn’t/weren’t…
- 疑问：Was/Were + S…?
