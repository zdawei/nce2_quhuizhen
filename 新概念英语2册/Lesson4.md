# exciting & excited
- exciting adj. 令人兴奋的
- excited adj. 感到兴奋的
- frighten vt. 使害怕
- frightening adj. 令人害怕的
- frightened adj. 感到害怕的
- The mouse frightened me.
- The mouse was frightening. I was frightened

- I frightened the mouse. 
- I was frightening.
- The mouse was frightened

1. 现在分词表示 “令人……的”；
2. 过去分词表示 “感到……的”。

# 现在完成时
## 构成
- 肯定: S. + have / has + done
- 否定: S. + haven’t / hasn’t + done
- 疑问: have / has + S. + done


## 现在完成时的使用条件

1. 表示一直延续到现在的动作。
2. 动作在过去已经发生，但对现在的影响依然存在，而且具体发生时间不明，一旦有具体发生的时间，就必须用一般过去时。

## 现在完成时的标志词

句中出现下列时间状语常常提示我们应该
使用现在完成时：
just( 英 式 ), already, yet, ever, never,
lately, recently, so far, up to now, before,
for..., since...,...

# 同位语 (appositive)
如果一句话中两个句子成分所指内容相同，则后者称为前者的同位语。

- John Smith, a friend of mine, has just called.

# 非延续性动词

不可延续性动词不能与表示一段时间的状语连用。
die, marry, come, arrive, leave, go,
finish, buy，。。。

# already & yet

- 在肯定句中用“already”多位于句中; (已经)
- 在否定和疑问句中用 “yet”, (否定句中译为“还 (没有) …”) 多位于句尾。

# has/have gone / been to
- has / have gone to … 去了……（还没回来）
- has / have been to … 去过 ……（已经回来）

# before 
before adv. 以前（单独使用，位于句尾，现在完成时标志词之一）

# ago
ago adv. …以前（不能单独使用，用于一般过去时）

# find + 宾语 + 宾补
find + 宾语 + 宾补: 觉得/认为某人某物……
- I find the book very useful.
